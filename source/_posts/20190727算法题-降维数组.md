## 算法题目：

已知如下数组：

```js
var arr = [[1, 2, 2], [3, 4, 5, 5], [6, 7, 8, 9, [11, 12, [12, 13, [14] ] ] ], 10];
```
<!--more-->

### 要求1： 将多维数组降成一维数组

#### 方法一： es6的flat()

```js
arr.flat(3);
```

#### 方法二： 投机取巧

```js
arr.toString().split(',');
```

或者

```js
arr.join().split(',');
```

####  方法三： 递归

```js
function flat (arr) {
	let result = [];
	function _flat (arr) {
		arr.map(function (value, index) {
			if (Array.isArray(value)) {
				_flat(value)
			}else {
				result.push(value)
			}
		})
	}
	_flat(arr)
	return result;
}
flat(arr)
```

简写：

```js
let flat = arr => [].concat(...arr.map(v => Array.isArray(v) ? flat(v) : v))
```
